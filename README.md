## Overview

This repository serves as a practical guide for learners looking to deepen their understanding of Entity Framework Core in .NET. You will find various exercises that focus on:

- Advanced query techniques.
- Implementing a rich domain model.
- Managing transactions effectively.
- Adopting best practices in software architecture.
- Intercepting SQL commands for better performance and debugging.

## Topics Covered

This course covers a wide range of topics, including:

- **Alura**: The platform hosting the course.
- **Architecture**: Understanding architectural patterns.
- **C#**: Programming language used.
- **.NET**: Framework utilized for building applications.
- **EF Core**: The core technology for data access.
- **Entity Framework**: ORM for working with databases.
- **Interception**: Techniques for intercepting SQL commands.
- **Query Optimization**: Improving query performance.
- **Rich Model**: Building complex domain models.
- **Transactions**: Managing data integrity during operations.

## Getting Started

To get started with this repository, you will need to have the following prerequisites:

- .NET 8 SDK installed on your machine.
- A basic understanding of C# and Entity Framework Core.
- Visual Studio or any preferred IDE for C# development.

## Installation

Follow these steps to set up the project on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shrutibankar/freelando_webapi_course-alura-entity-framework-core-transactions_part-5_dotnet-8_csharp-12.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd freelando_webapi_course-alura-entity-framework-core-transactions_part-5_dotnet-8_csharp-12
   ```
3. **Restore the NuGet packages**:
   ```bash
   dotnet restore
   ```

## Usage

Once the project is set up, you can run the application using the following command:

```bash
dotnet run
```

You can also execute the exercises provided in the repository to practice your skills. Check the `Exercises` folder for detailed instructions on each exercise.
